## What Irminsul covers

Irminsul watches several redemption-code sources and posts only newly discovered codes to subscribed channels. It also provides protected member spam reporting, audit logging, exact-permission moderation commands, and an opt-in anti-raid system for communities that need more than code announcements.

The bot supports **Genshin Impact**, **Honkai: Star Rail**, **Zenless Zone Zero**, **Honkai Impact 3rd**, **Neverness to Everness**, and **Wuthering Waves**.

:::note[Message-based commands]
Stoat does not provide Discord-style slash-command interactions. Irminsul reads normal messages beginning with `/`. Command names are case-insensitive, but IDs and other arguments retain their original casing.
:::
