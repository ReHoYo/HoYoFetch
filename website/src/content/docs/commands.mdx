Command names are case-insensitive. Examples use the default `/` prefix. Arguments such as channel IDs, member IDs, case IDs, and reasons are preserved exactly as entered.

:::caution[These are message commands]
Stoat does not expose Discord-style command interactions. Type the complete command as a normal message in a server channel.
:::
